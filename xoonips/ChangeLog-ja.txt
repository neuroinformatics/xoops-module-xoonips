3.45 リリース (2011-03-03)
+ 3.44 における不具合修正の為のメンテナンスリリース

(変更履歴)
* バグ修正
  - 添付ファルのダウンロード時の Content-Type ヘッダの問題修正
  - 3.24 からのアップデート時のバグ修正
  - アイテムが削除された時に参照していたアイテム側の Related To 項目が
    削除されない不具合の修正
  - Zip ファイルの展開に失敗する場合がある不具合修正
  - Add Item に関する不具合修正
    - 他人のアイテムを自身のプライベートに追加できてしまう
    - 公開承認作業中のアイテムを指定するとアイテムの編集ロックが外れなく
      なる
  - OAI-PMH リポジトリの setSpec に関する不具合修正
    - 各メソッドの応答のレコード中に現れる setSpec 値が ListSets 応答の
      setSpec 値と一致しない場合がある
    - 各メソッドの set 引数による選択的ハーベスティングの結果一覧が正しく
      ない場合がある
  - 特定の条件下でインポートに失敗する場合がある不具合の修正
  - その他累積的な不具合の修正
* 互換性の向上
  - Windows の IIS における互換性向上
  - PHP 5.3 における互換性向上
    - 警告メッセージの抑止
    - 非推奨な関数の撤廃(ereg, splitなど)
    - Fileinfo 拡張のマジックファイル設定が空の場合の動作対応
  - MySQL のオプションに STRICT_TRANS_TABLES が指定されている場合インス
    トールに失敗するアイテムタイプモジュールがある．
* 機能拡張
  - item_id による検索
